Core Web Vital:
Google consider to rank different web-pages

1. Cumulative Layout Shift (CLS)
In web performance, CLS is a Core Web Vitals metric used by Google to measure visual stability. It calculates how much unexpected movement of page content occurs while a page is loading (e.g., text suddenly shifting down when an ad or image loads).
Target Score: A good CLS score is ≤ 0.1, while anything over 0.25 is considered poor.
Optimization: You can track and debug your site's scores using the Google Search Console.

2. Largest Contentful Paint (LCP)
LCP reports the render time of the largest image, text block, or video visible in the viewport, relative to when the user first navigated to the page.
To provide a good user experience, sites should strive to have Largest Contentful Paint of 2.5 seconds or less.

3. First Input Delay (FID)
FID measures the time from when a user first interacts with a page (that is, when they click a link, tap on a button, or use a custom, JavaScript-powered control) to the time when the browser is actually able to begin processing event handlers in response to that interaction.
To provide a good user experience, sites should strive to have a First Input Delay of 100 milliseconds or less.
